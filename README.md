# BDSM恋爱倾向测试

一个精美的BDSM性格测试网站，采用浪漫粉色系设计风格，界面简洁优雅。

## 功能特点

- **50道专业测试题目** - 全面评估你在BDSM关系中的倾向
- **5个核心维度** - Dom(支配者)、Sub(顺从者)、Sadist(施虐者)、Masochist(受虐者)、Switch(切换者)
- **详细结果分析** - 包括性格特质、建议和玩法推荐
- **精美雷达图** - 可视化展示你的BDSM倾向分布
- **响应式设计** - 完美适配PC和移动设备
- **隐私保护** - 所有数据仅存储在本地浏览器中

## 测试维度说明

### 支配者 (Dominant)
喜欢掌控和引导关系的发展，享受制定规则和发号施令。

**推荐玩法：** 24/7权力交换、规则制定、训练调教、宠物play等

### 顺从者 (Submissive)
享受被引导和照顾的感觉，愿意将控制权交给信任的伴侣。

**推荐玩法：** 服从训练、规则遵守、臣服仪式、被束缚等

### 施虐者 (Sadist)
从给予伴侣身体或心理刺激中获得快感。

**推荐玩法：** 恋痛游戏、温度play、击打类刺激、语言羞辱等

### 受虐者 (Masochist)
从接受一定程度的刺激中获得快感和愉悦。

**推荐玩法：** 恋痛接受、被击打、温度刺激、羞辱play等

### 切换者 (Switch)
能够灵活切换不同角色，享受多样化体验。

**推荐玩法：** 角色切换、双向权力交换、多样化探索等
